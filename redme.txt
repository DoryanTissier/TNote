# TNote

## Description
TNote est une application web conçue pour la gestion des notes des étudiants. L'application permet aux enseignants de saisir et de gérer les notes des étudiants, aux étudiants de consulter leurs notes et leurs moyennes, et aux administrateurs de gérer les utilisateurs et les ressources.

- Gestion des notes par les enseignants (ajout, modification, suppression)

- Visual Studio Code
- XAMPP
- MySQL

## Installation

Clonez le repository :
```bash
git clone https://github.com/votre-utilisateur/tnote.git
cd tnote
```

Configurez la base de données :
1. Créez une base de données MySQL nommée `tnote`.
2. Importez le fichier SQL fourni pour créer les tables nécessaires :
```bash
mysql -u votre-utilisateur -p tnote < /path/to/tnote.sql
```

Configurez les variables d'environnement :
Créez un fichier `config.php` à la racine du projet et ajoutez les configurations suivantes :
```php
<?php
define('DB_HOST', 'localhost');
define('DB_USER', 'votre-utilisateur');
define('DB_PASS', 'votre-mot-de-passe');
define('DB_NAME', 'tnote');
?>
```

Déplacez les fichiers du projet dans le répertoire racine de votre serveur web (ex: /var/www/html/).  
Accédez au site via votre navigateur : `http://localhost/tnote`

## Contribuer
Pour contribuer au projet, veuillez suivre ces étapes :

1. Fork le projet :
   - Accédez à la page GitHub du projet.
   - Cliquez sur le bouton "Fork" en haut à droite pour créer une copie du projet sous votre compte.

2. Clonez votre fork :
   ```bash
   git clone https://github.com/votre-utilisateur/tnote.git
   cd tnote
   ```

3. Créez une branche :
   ```bash
   git checkout -b feature/NouvelleFonctionnalité
   ```

4. Faites vos modifications dans le projet.

5. Committez vos modifications :
   ```bash
   git add .
   git commit -m 'Ajout de la nouvelle fonctionnalité'
   ```

6. Poussez la branche :
   ```bash
   git push origin feature/NouvelleFonctionnalité
   ```

7. Ouvrez une Pull Request :
   - Allez sur la page GitHub de votre fork.
   - Cliquez sur "Compare & pull request".
   - Décrivez vos modifications et envoyez la Pull Request.
